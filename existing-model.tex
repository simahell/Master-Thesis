\section{セキュリティモデル}
\label{sec:SecurityModel}
セキュリティモデルは形式手法における入力にあたり、検査対象のシステムを\textcolor{red}{
命題論理
}
を用いて表現したものである。
セキュリティモデルに記述する項目は主に以下の三つである。
\begin{itemize}
\item 対象のシステムの構造と動作
\item 脅威モデル（想定される攻撃者の能力）
\item 安全性要件（安全上満たしているべき条件）
\end{itemize}
